62 /*
63 Macro to generate a generic selection association list element.
64 
65 Each element of the var args will be of form `(Type, expr)` - each element, a "tuple".
66 
67 Mapping `IT_GEN_ASSOC` over such an element results in `IT_GEN_ASSOC((Type, expr))`, which is then fed into
68 `IT_GEN_ASSOC_` without the extra parens- `IT_GEN_ASSOC_ (Type, expr)`.
69 
70 `IT_GEN_ASSOC_`, then creates the assoc list element - `Iterable(Type): (expr)`. A valid element to be used in the
71 `_Generic` selection assoc list.
72 */
